# Мессенджер

Современный веб-мессенджер с плавными анимациями и продвинутыми настройками.

## Возможности

- ✅ Регистрация и вход
- ✅ Чаты с сообщениями
- ✅ Голосовые и видеозвонки (демо)
- ✅ Темная/светлая тема
- ✅ Продвинутые настройки
- ✅ Плавные анимации
- ✅ Поиск по чатам
- ✅ Адаптивный дизайн

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub
2. Загрузи все файлы
3. Перейди в Settings → Pages
4. Выбери ветку `main` и папку `/ (root)`
5. Сохрани и дождись деплоя

Твой мессенджер будет доступен по адресу: `https://твой-username.github.io/название-репозитория`

## Использование

1. Открой `auth.html` для регистрации
2. Создай аккаунт
3. Войди в систему
4. Начни общаться!

## Технологии

- HTML5
- CSS3 (анимации, переходы, темы)
- Vanilla JavaScript
- LocalStorage для хранения данных

## Настройки

- Переключение темы (светлая/темная)
- Уведомления
- Звук сообщений
- Включение/выключение анимаций
